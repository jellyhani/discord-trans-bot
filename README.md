# 🌌 Jellybot (젤리봇) - 통합 번역 및 AI 컴패니언

**Jellybot**은 고성능 AI(GPT-5 / GPT-4.1 mini) 기반의 실시간 디스코드 번역 봇이자, 사용자의 대화를 학습하고 다양한 도구를 활용하는 스마트 멘토 시스템입니다. 

---

## 🚀 주요 기능

### 🌐 통합 번역 시스템 (Unified Translation)
문맥을 파악한 정확한 번역과 한국어 특화 교정 기능을 제공합니다.
- **실시간 자동 번역**: 언어 감지 + 오타 교정 + 번역을 단 한 번의 호출로 최적화.
- **시각 지능(Vision)**: 이미지나 스크린샷 내의 텍스트를 인식하여 자연스러운 번역 제공.
- **반응형 번역**: 메시지에 국기 이모지를 달아 즉각적으로 수동 번역 요청 가능.
- **한국어 오타/구어체 교정**: 야민정음, 신조어, 오타 등을 올바른 문장으로 선교정 후 번역.

### 🤖 Jellybot 멘토 엔진 (AI Mentor)
단순 답변을 넘어 실시간 데이터와 도구를 활용해 사용자를 돕습니다.
- **대화 내역 기억**: 대화 세션 및 히스토리를 관리하여 문맥에 맞는 답변 유지.
- **지능형 도구 연동 (Mentor Tools)**:
  - 🌤 **실시간 날씨**: 전 세계 주요 도시의 날씨 및 특수 기상 정보 조회.
  - 📈 **금융/경제**: 실시간 주식 시세 및 환율 데이터 분석.
  - 🔍 **웹 검색**: 최신 정보를 위한 실시간 검색 엔진 활용.
  - 🎮 **게이밍 프로필**: VRChat, Steam, Discord의 심층 프로필 및 활동 정보 분석.
- **페르소나 자동 학습**: 사용자의 취미, 관심사, 대화 스타일을 학습하여 개인화된 경험 제공.

### 🎨 페르소나 분석 및 시각화 (Persona Visualizer)
사용자의 대화 데이터를 정밀 분석하여 디지털 자아를 시각화합니다.
- **심층 보고서**: 사회적 행동, 관심사, 성장 과정을 분석한 예술적 보고서 생성.
- **AI 아바타 생성**: 분석된 페르소나를 바탕으로 DALL-E 3를 이용해 고유한 이미지 생성.

### 🛠 관리 및 통계 시스템
- **역할 기반 언어 동기화**: 특정 역할을 가진 유저에게 미리 설정된 언어를 자동으로 부여.
- **상세 통계 시각화**: `/serverstats`를 통해 서버 사용량 및 유저 순위 그래프 확인.
- **예산 및 비용 관리**: 월간 API 사용 예산을 설정하고 실시간으로 비용을 모니터링.

## 📁 프로젝트 구조

- `bot.py`: 봇의 진입점 및 확장 모듈 로드.
- `core/`: 핵심 로직 (번역 엔진, 멘토 엔진, 라우팅 시스템).
- `database/`: SQLite DB 기반의 유저 설정, 페르소나, 로그 및 데이터 관리.
- `cogs/`: 기능별 모듈 (admin, mentor, persona, slang 등).
- `utils/`: 외부 API 연동(날씨, 금융, 검색) 및 시각화 도구.
- `scripts/`: 데이터베이스 마이그레이션 및 관리용 유틸리티.
- `data/`: 시스템 프롬프트 및 도구 명세 관리 (JSON).

## 🛠 설치 및 실행

1. **환경 변수 설정**: `.env` 파일을 관리 폴더에 생성합니다. (`.env.example` 참고)
   ```env
   DISCORD_BOT_TOKEN=your_token
   OPENAI_API_KEY=your_key
   DISCORD_GUILD_ID=your_server_id
   ```
2. **의존성 설치**:
   ```bash
   pip install -r requirements.txt
   ```
3. **봇 실행**:
   ```bash
   python bot.py
   ```

## 🎮 주요 명령어

### 유저용 (Common)
- `/translate`: 수동 번역 요청.
- `/status`: 본인의 설정 및 상세 사용량/비용 통계 확인.
- `/draw-me`: 대화 데이터를 분석하여 페르소나 이미지와 보고서 수신.
- `/languages`: 지원하는 언어 목록 보기.

### 대화 세션 관리 (Session)
- `/chats`: 본인의 대화 세션 목록 조회.
- `/new_chat`: 새로운 대화 세션 시작.
- `/switch`: 기존 세션으로 전환.
- `/reset /archive /restore`: 세션 초기화 및 보관 설정.

### 관리용 (Admin - `서버 관리` 권한 필요)
- `/setlang`: 특정 유저나 역할의 번역 언어를 일괄 설정.
- `/serverstats`: 서버 전체의 통계 요약 및 사용량 차트 조회.
- `/setvision`: 이미지 번역용 모델 및 트리거 접약 설정.
- `/syncroles`: 서버 유저의 역할을 스캔하여 언어 설정을 자동 동기화.
- `/optimize`: 데이터베이스 공간 최적화 수행.
